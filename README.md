# 処方箋枚数予測アプリ

薬局の過去の処方箋枚数データをもとに、今後7日間・30日間の処方箋枚数を予測するStreamlitアプリです。

## 機能

- CSVアップロード
- 日付と処方箋枚数の自動整形
- 曜日、月、日、祝日、月初、月末の特徴量生成
- 7日前、14日前、28日前のラグ特徴量生成
- 7日移動平均、14日移動平均の特徴量生成
- RandomForestRegressorによる予測
- 過去データと30日予測のグラフ表示
- 7日予測表、30日予測表の表示
- 30日予測CSVと学習済みモデルのダウンロード

## 必要ライブラリ

```bash
pip install -r requirements.txt
```

## 起動方法

```bash
streamlit run app.py
```

## CSV形式

最低限、以下の2列が必要です。

```csv
date,prescription_count
2025-01-01,42
2025-01-02,58
```

## データ処理

- `date` は日付として読み込みます。
- `prescription_count` は数値として扱います。
- 日付が欠損している行は除外します。
- 処方箋枚数の欠損や日付の抜けは日次データに整形したうえで補完します。
- 同じ日付が複数ある場合は合計します。
- データが少ない場合は画面に警告を表示します。

## 出力

- 今後7日間の予測表
- 今後30日間の予測表
- 過去データと予測データのグラフ
- 30日予測CSV
- 学習済みモデル `prescription_model.joblib`
